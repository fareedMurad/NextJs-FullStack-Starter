## How to use this template? ðŸ¤”

To get started with this template, you just need to follow these simple steps:

1. Clone the repository:

   ```
   git clone https://github.com/fareedMurad/NextJs-FullStack-Starter.git
   ```

2. Install dependencies:

   ```
   npm i
   ```

3. Add `.env` file and add the values for the following keys:

   ```
    NEXTAUTH_URL=
    NEXTAUTH_SECRET=

    GOOGLE_ID=
    GOOGLE_CLIENT_SECRET=

    GITHUB_ID=
    GITHUB_CLIENT_SECRET=

    MONGODB_URI=
   ```

4. Run project locally

   ```
   npm run dev
   ```
